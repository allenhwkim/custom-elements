Bottom Navigation Bar

Material Design Spec   Documentation

Bottom navigation bars make it easy to explore and switch between top-level views in a single tap.

Installation

Just add the following script(< 2kb gzipped/polyfill in it) to your website. It will add CSS only once if required. 

<script defer src="https://unpkg.com/@custom-element/bottom-nav-bar"></script>
