Nav Item is an element used by a-menu, a-nav-drawer, a-bottom-nav-bar, and a-tab. It has three components; icon, text and shortcut.

Installation

Just add the following script(< 2kb gzipped/polyfill in it) to your website. It will add CSS only once if required. 

<script defer src="https://unpkg.com/@custom-element/nav-item"></script>
