## GridList
Grid lists are an alternative to standard list views.

[Material Design Spec.](https://material.io/guidelines/components/grid-lists.html#grid-lists-specs)

### Example
```
<mce-grid-list>
  <div>
    <img src="nature2.jpg">
    <div class="mce-text-with-icon">
      <h1>Single Line</h1>
    </div>
  </div>
  <div>
    <img src="nature2.jpg">
    <div class="mce-text-with-icon">
      <h1>Single Line</h1>
    </div>
  </div>
</mce-grid-list>
```
